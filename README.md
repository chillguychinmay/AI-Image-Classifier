# 🚀 AI Image Classifier

A deep learning project that classifies images into various categories using TensorFlow and Keras.

## 📌 Features
- Image preprocessing using OpenCV
- CNN model built with TensorFlow
- Real-time prediction support

## 🧠 Tech Stack
- Python 🐍
- TensorFlow
- NumPy
- Matplotlib
